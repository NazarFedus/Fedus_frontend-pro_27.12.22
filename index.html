Git commands:

  • git init - ініціалізуємо репозиторій
  • git status - перевіряємо статус репозиторію (які файли змінені, які видалені, які нові)
  • git add . - додаємо всі нові файли та зміни у файлах
  • git commit -m "message" - створюємо коміт з повідомленням
  • git log - побачимо історію комітів
  • git branch - побачимо всі гілки
  • git branch name - створюємо нову гілку
  • git checkout - переключаємося між гілками
  • git push - відправляємо зміни на сервер (github), щоб це зробити - потрібно все закомітити
  • git pull - отримуємо зміни з сервера на локальний репозиторій
  • git merge - зливаємо гілки
  • git clone - клонуємо репозиторій
  • git remote -v - побачимо всі віддалені репозиторії, в які ми можемо відправляти зміни
